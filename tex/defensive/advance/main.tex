\subsection{Basic Rubens: simple overcall non-competed}

We use Rubens' advances after a simple overcall to an 1-level suit opening lower than or equal to 2D. 
We first discussed the case where no interference, including X.
The rule is: transfer from opp's suit (skipping 2N), until 2X or 3X (direct raise).

\subsection{simple overcall doubled}

\begin{itemize}
    \item XX = lead-directing (Hx or Hxx)
    \item if we overcall 1X, then transfer from 1N instead. transfer to opp's suit = 10+ no good shape
    \item if we overcall 2X, then 2N = some art raise (see {\color{blue}the 2N raise})
    \item others = same
\end{itemize}

\subsection{simple overcall interfered}

the general rule is:
\begin{itemize}
    \item X = Responsive. but if they use some transfer, then X = LDX and cue = responsive
    \item P then X is still t/o (responsive)
    \item if mixed raise unavailable, then 2N = some art raise (see {\color{blue}1X\_interfered/the 2N raise})
    \item if direct cuebid still exists, then Rubens ON
    \item all bids become competitive and NF instead, only promises a weak-2 hand.
    \item others = same. i.e. jump is still invitational (of fit-showing)
\end{itemize}

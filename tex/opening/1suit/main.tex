\subsection{Gazzilli}

Before introducing opener's rebids, let's introduce Gazzilli, which uses \texttt{1H - 1SN - 2C \& 1S - 1N - 2C} to show 16+ (possibly with weak variant) hands. \\
For response: limit raise, \texttt{1H - 1S; 2C - 2S = 6+S inv}, and \texttt{1S - 1N; 2C - 3H = 1-S, 6+H inv} are still unaffected.
Otherwise, \texttt{2D} is art F that showing 8+ (GF against 16); while other bids are weak (7-) and natrual. (2N = pick-a-minor) \\
\texttt{\normalsize\color{brown} Quick tip for weak hand: usually bid 2M with 2+M (or even 1M if no good suit) since it is good opp 6M and not-too-bad opp strong 5M.}


For our minor opening, we can also employ Gazzilli thanks to transfer response \& unbal 1D saving space!
Specifically, \texttt{1C - 1DH - 1S \& 1D - 1M - 1N} are empty.
Notice that these are very suitable for Gazzilli, since it is a two-under bid for 2m. 
Therefore, we similarly define it to be either 16+ or weak unbal (i.e. the natrual 1m - 1M - 2m).
Responses are similar:

\begin{itemize}
    \setlength\itemsep{0pt}
    \item \texttt{+1 = art, 8+} (except the following) and others are nat and weak (2N = pick-a-minor)
    \item since \texttt{1m - 2H = 3-7 reverse Flannery}, \texttt{1D - 1S - 1N - 2H \& 1C - 1H - 1S - 2H} shows 8-10, 5S4H+
    \item others = (absolute) s/o
\end{itemize}

\subsection{1m - 1M}

In brief:
\begin{itemize}
    \setlength\itemsep{0pt}
    \item 4+M fit $\to$ always show first. (note: the only fit we won't show first round is GF fit after \texttt{1H - 1N*})
    \item \texttt{1m - 1M - 2m = 13-15, good 6m, 0-3M}
    \item \texttt{1m - 1M - 3m = 15-17, good 6m, 0-2M} (3M is a special case: Gaz first)
    \item all other unbal goes to Gaz, then ...
    \item if Gazzilli accepted (\texttt{... Gaz - +1}), back to 2m and 2M are nat min NF. (2M = 3M fit)
    \item if Gazzilli accepted (\texttt{... Gaz - +1}), boring 16+ hands just simply use \textit{the waiting bid}. otherwise, other bids usually shows shape and slam interest.
    \item if Gazzilli not accepted and 2M available, back to 2M shows 15-17, good 6m-3M. (ex: \texttt{1C - 1D; 1S - 2D; 2H}) jump 3M = GF, good 6m-3M.
    \item if Gazzilli not accepted and resp back to 2M, raise 3M or 2N = inv
    \item if Gazzilli not accepted, 3m is always 6+m and GF
\end{itemize}

\begin{itemize}

\subsection{fit-showing jump}

A jump is fit-showing (almost) iff PH. Showing 9+ cards and concentrated strength in the two suits, and
inv if not interfered, and pre to constr otherwise. In other senarios:

\begin{itemize}
    \setlength\itemsep{0pt}
    \item if we are opener, jump is weak (except 1M - X)
    \item if we are overcaller, jump is invitational
    \item if we are overcaller, and opp shows inv+, jump is weak and fit-showing
\end{itemize}

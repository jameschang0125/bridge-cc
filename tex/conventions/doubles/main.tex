\subsection{Lightner}

X against a voluntarily bid game/slam, if not slam-sacrifice or have showed power (thus PEN), is requesting a "strange" lead:

\begin{itemize}
    \setlength\itemsep{0pt}
    \item if LDX before, CANCELS IT
    \item if overcalled before, request DON'T LEAD
    \item against suit, usually have a void
    \item against jump to slam or NT, maybe have AK in a side suit
    \item lead dummy's suit
    \item lead declarer's suit
\end{itemize}
